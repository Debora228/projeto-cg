Objetivos

A proposta desse projeto é criar uma implementação dos 36 primeiros tutoriais de OpenGL que está disponível no tutorial, Introduction to OpenGL Graphics Programming, em Python.

Essa documentação visa ajudar futuras implementações do OpenGL em Python.

  • Requerimentos

    Nesse projeto será necessario as ferramentas a seguir.

    • keyboard_arrow_rightEditor de texto (Visual Code, Atom, Sublime)
    • keyboard_arrow_rightPython3.x
    • keyboard_arrow_rightPyOpenGL
    • keyboard_arrow_rightPacote FreeGLUT

    Preparando o Ambiente

    Debian/Ubuntu
    $ sudo apt-get install python3-pip
    $ pip3 install PyOpenGL PyOpenGL_accelerate
    $ sudo apt-get install freeglut3-d
  • Windows
    Python 3.x
    PyOpenGL

    $ cd Downloads
    $ pip install PyOpenGL-3.1.1-cp3x-cp3xm-win_amd64.whl

    Usando no codigo

    from OpenGL.GL import *
    from OpenGL.GLU import *
    from OpenGL.GLUT import *
